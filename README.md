# 🌿 kitker v3.1

> 영아 이사(Eve, 김영아)의 1인 워크스페이스.
> 노션 + 클릭업 + 구글캘린더 + 애플메모 통합, 1인 맞춤이라 70% 단순화.

## 🎯 핵심

- **Today 뷰가 메인** — 오늘의 일정·할일·시집·컨텍스트 한 화면
- **노션식 메모 트리 + 블록 에디터** — 무한 중첩, 슬래시 명령
- **메모 속성으로 일정/할일 토글** — 한 메모가 메모+일정+할일 동시
- **테마**: Petals (라이트) + Velvet Night (다크) 하이브리드

## 🛠 스택

Next.js 15 · React 19 · TypeScript · Tailwind v4 · shadcn/ui · Lexical · Neon Postgres + Drizzle · Vercel · PWA

## 📂 문서

- `KICK_OFF.md` — v3.1 최종 설계 (마크 방법론 7단계)
- `CLAUDE.md` — 개발 컨텍스트
- `TODO.md` — 진행 상황
- `CHANGELOG.md` — 변경 이력
- `docs/` — design-system / data-model / conventions

## 🚀 시작

```bash
git clone https://github.com/Direveyoung/kitker.git
cd kitker
cp .env.example .env.local
# .env.local 채우기
npm install
npm run dev
```
