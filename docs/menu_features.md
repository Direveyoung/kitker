## 🤖 클로이 챗 — 에이전트형 설계 (핵심)

### 컨셉
> "어디서든 사이드패널 열고 자연어로 던지면, 클로이가 직접 데이터 조작."

Claude 데스크탑 앱처럼 **항상 떠 있는 사이드패널**. 단순 검색·요약을 넘어서 **실제 액션**까지 수행.

### 화면 구성
```
┌──────────────────────────────┬──────────────────┐
│ (현재 보고 있는 모듈)         │ 🤖 클로이        │
│                              ├──────────────────┤
│                              │ [채팅 히스토리]  │
│                              │                  │
│                              │ 영아: 마운자로   │
│                              │ 다음주 일정 정리 │
│                              │                  │
│                              │ 클로이: 캘린더서  │
│                              │ 3건 찾았어요.    │
│                              │ ✅ 메모로 저장   │
│                              │                  │
│                              ├──────────────────┤
│                              │ [입력창]    [전송]│
└──────────────────────────────┴──────────────────┘
```

### 지원 액션 (Tool Use)

| 카테고리 | 자연어 예시 | 클로이 실행 |
|---|---|---|
| **쓰기** | "이거 메모로 저장해줘" | Notes에 새 노트 생성, 현재 컨텍스트 자동 포함 |
| **쓰기** | "오늘 투두에 김유진 회신 추가" | Todo 생성 |
| **쓰기** | "마운자로 프로젝트에 이 링크 넣어줘" | 해당 프로젝트 자료 추가 |
| **편집** | "이 메모 정리해줘" | 현재 노트 내용 분석 후 재구성 제안 → 승인 시 반영 |
| **검색** | "지난주 시집 관련 메모 다 보여줘" | Notes 필터 결과 패널에 |
| **요약** | "이번달 마운자로 진행상황 요약" | Project + Calendar + Todo 통합 요약 |
| **외부** | "이 기사 링크 찾아줘" | Web search → Reading List 저장 옵션 |
| **외부** | "이메일 초안 써줘" | 영아 톤·서명 자동 적용, 클립보드 복사 |
| **분석** | "이번달 개인 프로젝트 비용 얼마야" | Budget 집계 |
| **분석** | "Inbox 정리해줘" | 미분류 항목 분석 후 Todo/Notes/삭제 제안 |

### 컨텍스트 인식
클로이는 자동으로 다음을 인지:
- **현재 보고 있는 모듈** ("이 메모" = 현재 열린 노트)
- **선택된 항목** (텍스트 드래그 후 챗 던지면 그 부분 컨텍스트로)
- **최근 작업** (방금 본 5개 항목)
- **영아 이사 톤** (메모리 — 이브, 서명, "보고드립니다" 등)

### Tool Use 구현 (Anthropic API)
```
사용자 메시지
  ↓
Claude API (claude-sonnet-4-20250514)
  ↓
Tools 정의:
  - create_note(title, body, project_id?)
  - create_todo(text, due_date?, project_id?)
  - search_items(query, type?, date_range?)
  - update_note(id, body)
  - add_to_reading(url, note?)
  - fetch_url(url)  ← 웹 검색·링크 찾기
  - summarize_project(project_id)
  ↓
도구 실행 → 결과 → Claude → 사용자
```

### 모달 vs 사이드패널
- **데스크탑**: 우측 사이드패널 (`Cmd+L`로 토글), 항상 함께
- **모바일**: 하단 챗 버튼 → 풀스크린 모달
- **단축키**: 어떤 화면에서든 `Cmd+L` 또는 `/` 두 번 누르면 챗 입력 즉시

### 컨펌 UX
파괴적 액션 (삭제·대량 수정) → 항상 미리보기 + 승인 버튼
가벼운 액션 (메모 생성·투두 추가) → 즉시 실행 + 토스트 "되돌리기"

## 📦 Projects — "큰 그림 체크" 방식

### 갤러리 (1depth)
카드형 그리드, 각 프로젝트 아이콘·이름·핵심지표 1개.

```
┌──────────────┬──────────────┬──────────────┐
│ 🌿 eve.works │ 📦 kbc-dock  │ 💊 마슬라    │
│ Phase 1 진행 │ Live · 동의서│ 기획 단계    │
│ Todo 3 / Note│ 12건 수신    │ Note 5       │
├──────────────┼──────────────┼──────────────┤
│ 🏥 KOHBS     │ 📈 SuperStock│ 💉 마운자로  │
│ 컨셉 정리    │ KIS API 전환 │ 캠페인 진행  │
│ Note 2       │ Todo 7       │ Todo 12 / 캘 5│
└──────────────┴──────────────┴──────────────┘
                                + 새 프로젝트
```

### 프로젝트 상세 (2depth) — 자유 캔버스 + 위젯
ClickUp처럼 강제된 구조 X. **블록 추가 방식**.

영아 이사가 직접 위젯 배치:
- 📝 **메모 블록** — 자유 텍스트 (Lexical, 큰 그림·전략 메모)
- ✅ **투두 위젯** — 이 프로젝트 태그된 투두 자동 표시
- 📅 **일정 위젯** — 이 프로젝트 캘린더 항목
- 🔗 **링크 보드** — 자료·문서·외부 링크
- 🔑 **자격증명 블록** — 서비스명 / 아이디 / URL / 힌트 ("Bitwarden vault 'kbc-dock'")
- 📊 **숫자 위젯** — 핵심 지표 수동 입력 (인플루언서 N명, 계약완료 N건)
- 🖼 **이미지·스크린샷**
- 🤖 **클로이 요약 블록** — 자동 갱신 ("최근 7일 변화")

### 마운자로 예시 (영아 이사 케이스)
```
┌─ 💉 마운자로 캠페인 ─────────────────────┐
│                                          │
│ [📝 캠페인 큰 그림 메모]                 │

│ [✅ 마운자로 투두 (12)]                  │

│  - 릴리 가이드라인, 계약서 템플릿...     │
│                                          │
│ [🤖 클로이 주간 요약]                    │
│  지난 7일: 계약 2건 진행, 콘텐츠 1건...  │
```

## 📋 모듈별 핵심 기능 (요약)

### 🚀 Inbox
- 한 줄 입력 → Enter, 모바일 진입 시 자동 focus
- 분류 액션: → Todo / → Note / → Reading / → 프로젝트
- 클로이 챗: "Inbox 정리해줘" → 자동 분류 제안

### ✅ Todo
- 체크박스만, 자정 자동 carry-over
- 프로젝트 태그 가능 (선택)
- 클로이 챗: "내일 할 일 정리해줘"

### 📅 Calendar
- **Google Calendar 양방향 연동** (자체 구축 X)
- 보기: 월·주·일
- 프로젝트 태그 (이벤트 메타데이터에)
- 클로이 챗: "다음주 마운자로 일정 다 알려줘"

### 💭 Notes
- Lexical 에디터, 마크다운, 자동 저장
- 프로젝트 태그 (선택)
- 검색: Postgres tsvector (한글 형태소 처리 필요 — pgroonga 또는 ts_korean)

### 🔖 Reading
- URL 입력 시 메타데이터 자동 추출 (제목·썸네일·요약)
- 읽음/안읽음 토글
- 클로이 챗: "이 링크 찾아서 저장해줘"

### 📓 Journal
- 날짜별 한 페이지 (오늘 자동 생성)
- 프롬프트 옵션 (오늘 잘한 것 / 감사 / 시 영감 등)
- 클로이 챗: "이번주 일기 요약해줘"

### 💸 Budget
- 카테고리: 도메인 / 호스팅 / API / SaaS 구독 / 기타
- 프로젝트별 집계 (eve.works·kbc-dock·마슬라 각자 얼마)
- 월별·연간 뷰
- 영수증 메모 첨부 (이미지)
- 클로이 챗: "이번달 SuperStock 비용 얼마야"

### ⚙️ Settings
- Account / Appearance / Google 연동 / Data export / Chloe API 키
- 시집(Poetry) on/off 토글 (Phase 2 이후 분리 시)
